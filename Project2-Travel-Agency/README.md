# Travel Bliss – Responsive Travel Website:
Travel Bliss is a modern and fully responsive travel agency website built using HTML and CSS.  
This project was created as Project 2 for the DecodeLabs Frontend Development Internship Program.
The website showcases beautiful Indian travel destinations with a clean UI, responsive layouts, interactive sections, and mobile-friendly navigation.

# Features:
- Responsive Web Design
- Sticky Navigation Bar
- Hamburger Menu for Mobile Devices
- Hero Section with Background Image
- Popular Destinations Section
- Travel Packages Section
- About Section
- Footer Section
- Smooth Scrolling Navigation
- Hover Effects and Transitions
- Mobile-First Responsive Layout
- Flexbox and CSS Grid Usage

# Tech Stack:
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Font Awesome Icons
- Google Fonts

# Responsive Design:
This project is fully responsive and adapts to:
- Mobile Phones
- Tablets
- Laptops
- Desktop Screens
Media queries are used to ensure proper layout, spacing, typography, and navigation across different screen sizes.
