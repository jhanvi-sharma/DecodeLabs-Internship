Fashion Store Website: 
- A modern and stylish Fashion Store Landing Page built using HTML5 and CSS3 during my Frontend Development Internship at DecodeLabs. This project focuses on creating a visually appealing and responsive user interface with smooth layouts, elegant styling, and mobile-friendly design.

Project Highlights:
- Clean and modern fashion website UI
- Fully responsive design for different screen sizes
- Attractive hero banner section
- Stylish product showcase cards
- Smooth hover effects and transitions
- Organized and beginner-friendly code structure
- Soft pink aesthetic theme for an elegant user experience

Features:
- Responsive Navigation Bar
- Hero Section with Fashion Banner
- Featured Products Section
- About Brand Section
- Interactive Hover Animations
- Mobile Responsive Layout
- Clean UI/UX Design

Tech Stack: 
- HTML5
- CSS3
- Google Fonts
- Flexbox
- Responsive Web Design

Project Structure:
Fashion-Store/ 
  │── index.html 
  │── style.css 
  │── images/ 
  │── README.md

What I Learned Through this project, I improved my understanding of:
- Structuring webpages using HTML
- Styling modern interfaces with CSS
- Using Flexbox for layouts
- Creating responsive designs
- Adding hover effects and animations
- Organizing frontend project files
- Managing projects using GitHub
